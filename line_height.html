Sometimes we have to increase or decrease the length of the text of the displayed text. By using a style of CSS, the lines can be increased or reduced. If any style is not added, the length of each line is empty then is displayed as default.
CODE:
<!DOCTYPE html>
  <html lang="en-US">
   <meta charset="UTF-8">
    <style type="text/css">
    div{
      line-height:50%;
        }
    </style>
    <body>
      <div>
    	  Hello! I am Muntasir.This is an example of changing line height.The line height now increased by 50%.
      </div>
    </body>
  </html>

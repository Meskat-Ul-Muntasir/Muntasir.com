Has the width of the text ever been smaller than the width of the page, as shown in the picture below? If so, you have come to the right place. On this page you will find the solution to your problem.
WHY IT HAPPENS??
If the width of any other image or text or anything else on the page that contains the text is greater than 100% or the width of the original page, then this happens. For this you either need to reduce the width of the image or text with more than 100% width to 100%, or you have to increase the width of other text. To do this, go to the CSS file associated with the HTML file and you have to increase or decrease the width. The code that needs to be typed is mentioned below.
CODE:
<div style="width:100%;">Hello! I am Meskat UL Muntasir.I live in Rajshahi,Bangladesh.</div>
Or,
(In CSS file)
div{
width:100%;
}
OUTPUT
বাংলায়
কখনো কি উপরে দেখানো ছবির মত টেক্সট এর প্রস্থ পেজের প্রস্থের থেকে ছোট হয়ে গেছে? যদি হয়ে থাকে তাহলে আপনি একদম সঠিক জায়গায় এসেছেন।এই পেজেই আপনি আপনার সমস্যার সমাধান পেয়ে যাবেন।
কেন এমন হয়??
যেই পেজে ওই টেক্সটটি আছে সেই পেজে যদি অন্য কোন ইমেজ অথবা টেক্সট অথবা অন্যকিছুর width ১০০% বা মুল পেজের width এর থেকে বেশি হয়ে থাকে, তাহলে এমনটি হয়ে থাকে। এজন্য হয় আপনাকে ১০০% এর বেশি width সম্পন্ন ইমেজ অথবা টেক্সট এর width কে ১০০% এ কমিয়ে আনতে হবে, আর না হয় অপর টেক্সটের width কে বাড়িয়ে নিতে হবে। এজন্য এইচটিএমএল ফাইল টি এর সাথে সম্পৃক্ত সিএসএস ফাইল এ গিয়ে আপনাকে width কে বাড়িয়ে অথবা কমিয়ে নিতে হবে।এজন্য যে কোডটি টাইপ করতে হবে সেটি নিচে উল্লেখ করা হলো।

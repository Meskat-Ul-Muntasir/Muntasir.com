FACTORIZING PROGRAM(LINES:14)

When any integer number is input in this software, the software collects all the factors of the number in a list. The main engine of this software is a while loop which starts dividing the number input by all the numbers starting from 1. An if....else block collects the number in the list only when a number divides the input number.Since the program divides the input number by all the numbers from one to the input number, it takes a little time if a large number such as 10 million is input. 

ask_factorizable_num=int(input('Factorizable num:'))
factors=[ ]
i=1
while i<= ask_factorizable_num:
  if ask_factorizable_num % i ==0:
    factors.append(i)
  i+=1
print('Factors of ',ask_factorizable_num,'=',factors)
total=len(factors)
print('Total factors of',ask_factorizable_num,':',total)
if total== 2:
  print('Details:',ask_factorizable_num, 'is a basic(মৌলিক সংখ্যা) number')
else:
  print('Details:',ask_factorizable_num,'is not a basic (মৌলিক সংখ্যা) number')

In HTML, sometimes information needs to be placed in the middle of the table. On this page you will find how to create a table in HTML. A number of tags are required to create a table. These are explained below.
"table" tag(<table></table>)
One of the most important for creating a table is the table tag (<table> </ table>).
CODE:
<table>
</table>
"tr" tag(<tr></tr>)
Imagine you have a table. There are several rows of cells along the length and width of the table where different information is contained. Each row along the width of the HTML is called a table row. The tr tag (<tr> </tr>) is used to create each of these rows. And all the text included in the tr tag will be displayed as a row or line.
CODE:
<table>
  <tr>
   Muntasir
   Nothing
  </tr>
  <tr>
   Jamee
   Jama
  </tr>
  <tr>
   Orjon
   Nastik
  </tr>
  <tr>
   Salahuddin
   Biggani
  </tr>
  <tr>
   Prottoy
   Lo*i
  </tr>
</table>
"td" tag(<td></td>)
Notice that each table has a column with a row. That is, there is one or more information in each column. This data from Row is called table data. The td (<td> </td>) tag is used to display data in a table.
CODE:
<table>
  <tr>
   <td>Muntasir</td>
   <td>Nothing</td>
  </tr>
  <tr>
   <td>Jamee</td>
   <td>Jama</td>
  </tr>
  <tr>
   <td>Orjon</td>
   <td>Nastik</td>
  </tr>
  <tr>
   <td>Salahuddin</td>
   <td>Biggani</td>
  </tr>
  <tr>
   <td>Prottoy</td>
   <td>Lo*i</td>
  </tr>
</table>
"thead" tag(<thead></thead>) and "tbody" tag(<tbody></tbody>)
The table of the table and the data was displayed but the table head was not displayed. Table Head did not understand what? I'm saying. To understand what kind of information on any site of the table or column is given some data in the top ROW. In the process of this data, it is called Table Head. For this, the tag is used to display table head's data. But the problem is that the rest will be displayed as? That's we have to give instructions to the computer for this also. The information you do not want to identify as the table, type in (<tbody> </tbody>) tag. In all, our final product will be somewhat like this.
CODE:
<table>
  <thead>
   <tr>
     <th>Name</th>
     <th>Nickname<th>
   </tr>
  </thead>
  <tbody>
    <tr>
     <td>Muntasir</td>
     <td>Nothing</td>
    </tr>
    <tr>
     <td>Jamee</td>
     <td>Jama</td>
    </tr>
    <tr>
     <td>Orjon</td>
     <td>Nastik</td>
    </tr>
    <tr>
     <td>Salahuddin</td>
     <td>Biggani</td>
    </tr>
    <tr>
     <td>Prottoy</td>
     <td>Lo*i</td>
    </tr>
  </tbody>
</table>
OUTPUT

ADDING SOME STYLES

The table was created but it was not made stylish. So, we will make it stylish now. So first we will put a border with it.
ADDING BORDER
Type the following code in your CSS file to insert the border.
CODE:
table,th,td{
border:1px solid black;
}
OUTPUT
BORDER COLLAPSE
Looking at the picture of the output above, you must have realized that it did not look the way we wanted to from the border. One thing is for sure! There is a problem free with every solution of programming. So let's move on to the solution of our problem now. We now need to use the border-collapse style to bring the borders together. The following code is displayed.
CODE:
table,th,td{
border-collapse:collapse;
}
OUTPUT
BORDER SPACING
Look, again a problem has appeared. The borders have been displayed in the size of the text, that is, very close to the text. For this we have to use padding style now. You must have understood by now why I am using padding style. So I did not say. The code is displayed below.
CODE:
th,td{
padding:5px;
}
CHANGING WIDTH
When you look at the output, you will notice that the table is not displayed across the entire screen. So now we have to increase its width. To do this, type the following code in the corresponding place of CSS.
CODE:
table{
width:100%
}
TEXT-ALIGN
Now we need to add a little more style. When the teble data is too large, it can move to the right of the cell. For this we need to add text-align style. The following code is given below.
CODE:
th{
text-align:left;
}
OUTPUT

ADDING BORDER
বর্ডার লাগানোর জন্য নিচের কোডটি আপনার সিএসএস ফাইল এ টাইপ করুন।

BORDER COLLAPSE
উপরের আউটপুটের ছবিটি দেখে নিশ্চয়ই বুঝতে পেরেছেন যে আমরা বর্ডার থেকে যেভাবে লাগাতে চেয়েছিলাম সেভাবে লাগেনি। সত্যিই একটা কথা সত্য !প্রোগ্রামিং এর প্রত্যেকটা সমাধানের সাথে একটা সমস্যা ফ্রী। তো যাই হোক এখন আমাদের সমস্যার সমাধান এর দিকে আসা যাক। বর্ডার গুলোকে একসাথে লাগিয়ে দেওয়ার জন্য আমাদেরকে এখন বর্ডার কলাপ্স(border-collapse:;) স্টাইল ব্যবহার করতে হবে। নিচের কোডটি প্রদর্শন করা হলো।
CODE:
table,th,td{
border-collapse:collapse;
}
OUTPUT
BORDER SPACING
দেখুন আর একটা সমস্যা হাজির হয়ে গেল।বর্ডার গুলো একদম টেক্সটের মাপে অর্থাৎ একদম টেক্সট এর সাথে ঘেঁষে ডিসপ্লে করা হয়েছে। এজন্য আমাদেরকে এখন প্যাডিং স্টাইল ব্যবহার করতে হবে। কেন প্যাডিং স্টাইল ব্যবহার করছি সেটা নিশ্চয়ই আপনি এতক্ষণে বুঝে গেছেন।তাই আর বললাম না। এর কোডটি নিচে ডিসপ্লে করা হলো।
CODE:
table{
padding:5px;
}
CHANGING WIDTH
আউটপুট টি দেখলে আপনি বুঝবেন যে টেবিলটি পুরা স্ক্রিনের প্রস্হ জুড়ে ডিসপ্লে করা হয়নি। এজন্য আমাদের এখন এটির width বাড়াতে হবে। এজন্য নিচের কোডটি সিএসএস-এর সংশ্লিষ্ট জায়গায় টাইপ করুন।
CODE:
table{
width:100%
}
TEXT-ALIGN
এবার আরও একটু স্টাইল যোগ করতে হবে। যখন টেবিল ডাটা অনেক বড় হবে, তখন সেটা সেল এর ডান দিকে চলে যেতে পারে। এজন্য আমাদেরকে text-align স্টাইল যুক্ত করতে হবে নিচেরটার কোডটি দেওয়া হল।
CODE:
th{
text-align:left;
}

Background image or color is very important to enhance the beauty of the website. These two functions are styles of CSS. The code required for this is presented below.

BACKGROUND IMAGE

CODE:
<!DOCTYPE html>
<html lang = "en-US">
 <meta charset = "UTF-8">
  <title> backgroundImage </title>
  <style type = "text/css">
   body {
   background-image: url("background.jpg");
    } 
   </style>
  <body>
   <h1>This is an example of setting background image.</h1>
  </body>
  </html>
DISCRIPTION
To set the background image you need to add " background-image: ;" style in CSS.In this case, you have to specify which file the background image is in. For this, you have to type the name of the image in the quotation with the URL. If this image is included in a file, then first type the name of the file with a slash sign (/) and then type the name of the image. But even more important is to select the place where the image wil be set as background . If you want to put the image as the background of a text, you must include the code with the class or the id of the tag. And if you want to put the image as the background of the whole page, then you must include the code in the body tag. Including the body tag will set the background image of the whole page.

BACKGROUND COLOR

CODE:
<!DOCTYPE html>
<html lang = "en-US">
 <meta charset = "UTF-8">
  <title> backgroundImage </title>
  <style type = "text/css">
  .back_color{
   background-color:blue;
   padding:10px 30px 10px 30px;
   border-radius:8px;
    } 
   </style>
  <body>
   <a href="####&"gt;<div class="back_color">This is an example of setting background color.</a>
  </body>
  </html>
DISCRIPTION
Multiple styles need to be added to CSS to set the background color. Three of these styles are enough to fix the background-color. These three are background-color :; , padding :; And border-radius :; . You need to know these three styles separately, but here I am very brief. First come the background-color : ; style. In this style you need to type the name of the color that you want to set as the background. But not all colors can be expressed in language so color is often indicated by hex color code or RGB. With this, many colors can be used very easily. You can find them on various websites by searching on Google. Now come to padding : ; style. Only because of the background color style, the background color will only cover as much space as the text. That's why many times it doesn't look so beautiful. How far the background color will extend from the text is the padding style that is indicated. The pixel unit is used as the unit of expansion. In this case, how wide it will be in four directions is indicated by four pixel lights in a row. The first one indecates the top, the second one indecates the right, the third one indecates the bottom and the fourth one indicates the left. The code above has a convenient pixel rate. Therefore to make it more stylish border-radius : ; tyle must be used. Here you have to type a pixel of the same radius as the radius of the background-color particles will be curved. As a result, the particles of the background-color will be curved.

ব্যাখ্যা
ব্যাকগ্রাউন্ড ইমেজ সেট করার জন্য আপনাকে সিএসএস এ background-image: ; স্টাইলটি যোগ করতে হবে। এক্ষেত্রে ব্যাকগ্রাউন্ড ইমেজটি কোন ফাইলে আছে এটি আপনাকে নির্দেশ দিতে হবে।এজন্য ইউআরএল দিয়ে ইমেজটির নাম কোটেশন এর মধ্যে টাইপ করতে হবে। যদি এই ইমেজটি কোন ফাইল এর অন্তর্ভুক্ত থাকে তাহলে প্রথমে ফাইলটির নাম টাইপ করে স্ল্যাস চিহ্ন(/) দিয়ে তারপর ইমেজটির নাম টাইপ করতে হবে। কিন্তু তার চেয়েও বেশি ইম্পর্টেন্ট হল যে ইমেজ টি কিসের ব্যাকগ্রাউন্ড হিসেবে লাগানো হবে সিটি নির্বাচন করা। যদি আপনি কোন টেক্সট এর ব্যাকগ্রাউন্ড হিসেবে ইমেজটি লাগাতে তাহলে অবশ্যই টেক্সটির ট্যাগ অথবা ট্যাগ এর ক্লাস আইডি জন্য কোডটি টাইপ করতে হবে। আর যদি আপনি পুরো পেজটির ব্যাকগ্রাউন্ড হিসেবে ইমেজটি লাগাতে চান তাহলে অবশ্যই অবশ্যই কোডটি বডি ট্যাগ এর অন্তর্ভুক্ত করতে হবে। বডি ট্যাগটির অন্তর্ভুক্ত করায় পুরো পেইজটি ব্যাকগ্রাউন্ড ইমেজ টি সেট হয়ে যাবে।

ব্যাখ্যা
ব্যাকগ্রাউন্ড কালার সেট করার জন্য সিএসএস এ একাধিক স্টাইল যুক্ত করতে হয়।এর মধ্যে তিনটি স্টাইল background-color ফিক্স করার জন্য যথেষ্ট। এ তিনটি হলো background-color:; , padding:; এবং border-radius:; । এ তিনটি স্টাইল গুলোর জন্যই আপনার আলাদা করে জানার প্রয়োজন তবে এখানে আমি অতি সংক্ষেপে বলছি। প্রথমে আসি background-color:; স্টাইল এর কাছে। এই স্টাইলে আপনাকে সেই রং টির নাম টাইপ করে দিতে হবে যেটি আপনি ব্যাকগ্রাউন্ড হিসেবে সেট করতে চান । কিন্তু সব রং ভাষায় প্রকাশ করা যায় না এজন্য অনেক সময় হেক্স কালার কোড অথবা আরজিবি দ্বারা রং কে নির্দেশ করা হয়। এর মাধ্যমে অনেক অনেক রং কে অতি সহজে ব্যবহার করা যায়। সেগুলো আপনি গুগলে সার্চ করার মাধ্যমে বিভিন্ন ওয়েবসাইটে পেতে পারেন। এবার আসি padding:; স্টাইল এর কাছে। শুধুমাত্র ব্যাকগ্রাউন্ড কালার স্টাইল যুক্ত করার কারণে ব্যাকগ্রাউন্ড কালার টি শুধুমাত্র টেক্সট টি যতটুকু জায়গা জুড়ে আছে সেই জায়গাটুকুই জুড়ে থাকবে। এজন্য অনেক সময় সেদিকে দেখতে অতটা সুন্দর মনে হয় না। ব্যাকগ্রাউন্ড কালার টি টেক্সট থেকে কত দূরে বিস্তৃত থাকবে এইটা প্যাডিং স্টাইল যারা নির্দেশ করা হয়ে থাকে। এ বিস্তারের একক হিসেবে পিক্সেল(px) একক ব্যবহার করা হয়। এক্ষেত্রে চার দিকে কতটা বিস্তৃত হয়ে থাকবে এটা পরপর চারটি পিক্সেল লাইট এর মাধ্যমে নির্দেশ করা হয়। প্রথমটি উপর, দ্বিতীয়টি ডান, তৃতীয় টি নিচ এবং চতুর্থটি বাম দিকে নির্দেশ করে।উপরের কোড টি তে সুবিধাজনক পিক্সেল রেট দেওয়া আছে।কিন্তু এখন সমস্যা হল ব্যাকগ্রাউন্ড কালার টি আয়তক্ষেত্র অথবা বর্গাকৃতির হবে। এজন্য এটিকে আরো স্টাইলিশ করার জন্য border-radius:; স্টাইল টি ব্যবহার করতে হবে। background-color টির কণাগুলো যতটা ব্যাসার্ধে বাঁকানো থাকবে সেই সেই ব্যাসার্ধের পিক্সেল একটি এখানে টাইপ করতে হবে।তার ফলে background-color টির কণাগুলো বাঁকানো হয়ে যাবে।
